Ladda - jQuery Version

A UI concept which merges loading indicators into the action that invoked them. Primarily intended for use with forms where it gives users immediate feedback upon submit rather than leaving them wondering while the browser does its thing. The jQuery version provides compatibility with older versions of Internet Explorer by using jQuery in place of things like document.querySelectorAll, element.addEventListener, etc.
